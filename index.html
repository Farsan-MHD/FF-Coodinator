ABOUT ME

Hello, I’m Mohammed Farsan, a budding web designer with a passion for creating beautiful and functional websites. Although I am at the beginning of my journey in the field, I am eager to learn, grow, and bring fresh, creative ideas to life through design.

I have always been fascinated by the way design can communicate a message and shape user experiences. This curiosity led me to explore web design, where I found a perfect blend of creativity and technical skills. I’m excited to keep expanding my knowledge and honing my abilities as I take on new challenges and projects.

My approach to design is simple yet thoughtful. I believe that every website should not only look great but also serve its purpose effectively, providing a seamless experience for users. I enjoy experimenting with colors, typography, and layouts to find the perfect balance that brings a brand’s vision to life

Even though I am just starting out, I am committed to continuously learning and staying updated with the latest trends and techniques in web design. I am confident that my enthusiasm, dedication, and attention to detail will allow me to create designs that stand out and make a difference.

If you’re looking for a fresh perspective and a designer who’s eager to learn and grow, I’d love to connect. Let’s create something amazing together!

HTML is the foundation of web design, allowing you to structure text, images, and links, and it works in combination with CSS and JavaScript for styling and interactivity.

CSS

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls the appearance of HTML elements, including colors, fonts, spacing, and positioning. CSS allows you to create visually appealing designs by separating the content (HTML) from the design.

85%

JAVASCRIPT

JavaScript is a programming language that adds interactivity and dynamic behavior to web pages. While HTML structures the content and CSS styles it, JavaScript makes the page responsive by handling user actions, updating content, and controlling multimedia.
